Projects

  • Identifying mode confusion in aircraft-pilot interactions Most human-machine interaction systems suffer from the dreaded failure of automation surprise. This project aims at predicting automation surprises in safety critical systems such as aircraft-autopilot system; and then proposing possible resolution methods to get rid of the mode mismatch events thus identified. | Collaborators : Prof. Meeko Oishi, Vignesh Sivaramakrishnan
  • State estimation subject to packet losses

    The project aimed at studying state estimation subject to random packet dropouts. Addressing different ways to incorporate this randomness led to two different subproblems:

    (i) packet losses are Markovian in nature: such packet losses are common in communication channels (such as ethernet).

    (ii) packet losses that occur within specific regions, where the regions are themselves distributed according to known priors (such as radar jammers).

    Systematic inclusion of known information about packet losses (Markovian, or state dependent distribution of jammers) leads to an improved accuracy in the baseline Kalman filter, with a comparable computation complexity. | Collaborator : J. S. Nandiganahalli
  • Consensus based hybrid state estiamtion for networked control systems (NCS) [code] This project aimed at extending the idea of consensus based state estimation (such as consensus-Kalman filters) to hybrid state estimators (such as IMM, GPB-I, GPB-II, etc.). The key idea is to allow consensus at each agent of the NCS and include it in a manner that allows for time-changing topologies of the connecting network. The state estimator developed outperforms all consensus based estimators for hybrid systems (as of 2018). | Collaborators : Raj Deshmukh, Cheolhyeon Kwon
